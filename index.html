Install on iPhone

  1. Open dondlingergc.com/calc/ in Safari.
  2. Tap the Share button.
  3. Tap Add to Home Screen.
  4. Launch it from your home screen like a normal app.
If you see an older broken version, clear the browser cache and remove the previous home-screen install before reinstalling /calc/.
